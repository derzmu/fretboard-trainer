# Griffbrett-Trainer

Interaktive Web-App zum Lernen von Skalen, Intervallen und Tonpositionen auf dem Gitarren-Griffbrett.

## Features

**Skalen** – Dur, Moll (natürlich/harmonisch/melodisch), Pentatoniken, Blues und alle Kirchentonarten in allen 12 Tonarten. Farbcodierte Stufen mit hervorgehobenem Grundton.

**Intervalle** – Kleine/große Terz, Quarte, Quinte, Septimen und Oktave visuell auf dem Griffbrett. Schnellwahl-Buttons für direkten Zugriff.

**Ton-Finder** – Alle Positionen eines einzelnen Tons auf dem gesamten Griffbrett mit Auflistung aller Saite/Bund-Kombinationen.

## Tech Stack

- React 18
- Vite
- GitHub Pages (via GitHub Actions)
